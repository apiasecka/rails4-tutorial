## Ostateczne terminy oddania projektów

Czasami muszę wstawić coś takiego na stronie głównej aplikacji:

    <script>
    $(function () {
     deadline = new Date(2011, 06, 22);
     $('#defaultCountdown').countdown({
      until: deadline,
      compact: true,
      layout: '<b>{dn}{dl} {hnn}{sep}{mnn}{sep}{snn}</b>',
      format: 'dHMS'});
    });
    </script>
